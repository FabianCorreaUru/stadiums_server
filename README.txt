1) Download the server into a folder called stadiums
2) Position in "stadiums/server" and run the server (npm run dev)
3) The server will be listening on port 5000 (http://localhost:5000/api/)
4) Download the client into a folder called stadiums
5) Position in "stadiums/client" and run the server (npm run start)
6) The client will be listening on port 3000 (http://localhost:3000/)
7) Enjoy the Stadium Geolocation App
